## Lessons to remember

**Git**
- **Golden rule:** always `git pull` at the start, `git push` at the end. Got lucky once today because Levi was working on different files in parallel.
- `git add` ≠ commit. Three steps: `add` → `commit -m "..."` → `push`. Skip one and Levi sees nothing.
- For features: `git checkout -b feature/<name>` → work → `gh pr create`. Main stays clean.

**Meta API**
- A System User needs **two** assets: ad-account and the app itself. Without the app link, token generation doesn't work.
- Minimum-scope principle: `ads_read` only. Not `attribution_read` (Meta has largely deprecated that product) and definitely no `*_management` scopes.
- Always pin the API version. "Latest" can break without warning.

**Tooling on Windows**
- `winget install <id>` works for Git CLI, Node, Python. No admin needed.
- New terminals required after install to refresh PATH.
- Python stdout in cp1252 codepage chokes on Unicode (`✓`, `—`). Fix: `sys.stdout.reconfigure(encoding="utf-8")` at the start of `main()`.
