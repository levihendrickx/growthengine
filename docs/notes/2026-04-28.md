## Lessen die ik wil onthouden

**Git**
- **Gulden regel:** altijd `git pull` aan het begin, `git push` aan het eind. Vandaag al een keer geluk gehad omdat Levi parallel werkte aan andere bestanden.
- `git add` ≠ commit. Drie stappen: `add` → `commit -m "..."` → `push`. Eén overslaan = niets bij Levi te zien.
- Voor features: `git checkout -b feature/<naam>` → werken → `gh pr create`. Main blijft schoon.

**Meta API**
- System User heeft **twee** assets nodig: ad-account én de app zelf. Zonder app-koppeling werkt token-generatie niet.
- Minimum-scope-principe: `ads_read` alleen. Niet `attribution_read` (Meta heeft dat product grotendeels uitgefaseerd) en zeker geen `*_management` scopes.
- Pin altijd de API-versie. "Latest" kan zonder waarschuwing breken.

**Tooling op Windows**
- `winget install <id>` werkt voor Git CLI, Node, Python. Geen admin nodig.
- Nieuwe terminals nodig na een installatie om PATH te verversen.
- Python stdout in cp1252 codepage stikt op Unicode (`✓`, `—`). Fix: `sys.stdout.reconfigure(encoding="utf-8")` aan het begin van `main()`.
