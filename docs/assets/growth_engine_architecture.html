03 Vision LLM — afbeelding naar tekst
GPT-4o Vision of Claude 3.5 Sonnet

Vision LLM — feitelijke beschrijving

Elke ad-afbeelding wordt eenmalig doorgestuurd. De LLM beschrijft alleen wat letterlijk zichtbaar is — geen interpretatie, geen hooks, geen oordeel. Prompt bevat expliciet verbod op aannames. Bij onduidelijke details: "niet zichtbaar" in plaats van raden.

"Vrouw van circa 30 jaar op strandbed. Zilveren armband aan linkerpols. Twee kinderen spelen op achtergrond in het zand. Zonnige dag, warme kleuren. Geen tekst zichtbaar in afbeelding."

merge op ad_id + utm_ad + datum

06 MeTTa / PLN — redenering
Hyperon — probabilistische logica

MeTTa / PLN — redeneert continu over alle atoms

Detecteert patronen · berekent confidence scores · signaleert kansen · leert van elke nieuwe campagnecyclus. Brand atom staat buiten de analyse — wordt pas actief bij output.

Voorbeeldpatronen die PLN detecteert

"lifestyle + weekend + winter" → ROAS 3.8 · conf: 0.79 · n=14

"urgency + feestdag -3 dagen" → CTR hoog maar conv laag · conf: 0.71

"Valentijn dag -10 starten" → 40% hogere ROAS dan dag -3 · conf: 0.84

"armband zilver + strand + zomer" → beste seizoenscombinatie · conf: 0.88

08 PLN — Ad specificaties ophalen
PLN query op Atomspace

PLN bepaalt wát er gegenereerd wordt

Gebruiker vraagt: "10 ads voor zilveren armband." PLN doorzoekt Atomspace op alle atoms die relevant zijn voor dit product, dit seizoen en deze dag. Sorteert op confidence score. Selecteert de sterkste bewezen combinaties én een set onderbelichte variaties met potentieel.

Wat PLN teruggeeft — 10 ad specs

Ad 1–4 · bewezen combinaties · conf >0.80 · lifestyle + model + weekend

Ad 5–7 · sterke variaties · conf 0.65–0.79 · lifestyle + product-shot

Ad 8–10 · onderbelicht potentieel · conf 0.50–0.64 · nieuwe combinatie testen

Per ad spec

Inhoud van één spec

  • visual_style · setting
  • hook_type · emotional_angle
  • product focus · context
  • dag/seizoen context
  • brand kleuren · tone
  • image_ref (beste match)
  • confidence score
  • basis-atoms (traceerbaar)
10 ad specs → LLM voor tekstgeneratie
09 LLM — Prompt generatie (alleen tekst)
Tekst generatie · geen creatieve beslissingen

LLM zet ad specs om naar image prompts

De LLM krijgt per ad een strak gestructureerde spec van PLN. Zijn enige taak: die spec vertalen naar een gedetailleerde image prompt. Alle creatieve keuzes zijn al gemaakt door PLN. De LLM vult alleen de taal in.

Voorbeeld: spec → prompt

Spec · lifestyle · strand · model · zilveren armband · warm-goud · zomer

Prompt · "Woman relaxing on sunlit beach, close-up of silver bracelet on wrist, warm golden tones, natural light, editorial style, soft background blur, luxury feel"

Tevens gegenereerd

Ad copy per ad

  • headline op basis van spec
  • body copy · tone of voice
  • CTA uit brand atom
  • alles traceerbaar naar spec
10 prompts → verificatieloop
10 Verificatieloop 1 — Prompt check
Geautomatiseerde check

Controle: volgt prompt de spec?

  • elk veld uit spec aanwezig?
  • geen toegevoegde elementen?
  • brand kleuren correct?
  • tone of voice correct?
  • geen hallucinaties in copy?
PLN variatie check

Controle: zijn variaties correct gestuurd?

  • elke prompt uniek?
  • variaties binnen PLN-kaders?
  • confidence-verdeling correct?
  • geen overlap tussen ads?
Bij fout

Afwijzing → hergeneratie

  • falende prompt terug naar LLM
  • met exact de afwijkingsreden
  • max 2 pogingen per prompt
  • bij blijvende fout: flag voor user
✓

Goedgekeurd — alle 10 prompts voldoen aan de spec. Geen hallucinaties. Variaties binnen PLN-kaders. Klaar voor image generator.

goedgekeurde prompts → image generator
11 Image generator
Generatie · bijv. DALL·E 3 of Midjourney API

Image generator ontvangt goedgekeurde prompts

Elke goedgekeurde prompt + image_ref (stijlreferentie uit Atomspace) worden naar de image generator gestuurd. De image_ref stuurt de stijl — de prompt stuurt de inhoud. Output: 10 afbeeldingen, één per ad spec.

Input per generatie

Wat de generator krijgt

  • gedetailleerde image prompt
  • image_ref als stijlreferentie
  • formaat · ratio (Meta specs)
  • brand kleurpalet als sturing
gegenereerde afbeeldingen → verificatieloop 2
